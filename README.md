# Lexikon

An ODBC approach for calling LLMs.

## Story Generators

```bash
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-claude.ts 1
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-claude.ts 2
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-groq.ts 1
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-groq.ts 2
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-openai.ts 1
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-openai.ts 2
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-gemini.ts 1
deno run --allow-net --allow-env --allow-read --allow-write examples/story/generate-gemini.ts 2
```
